# WakeTask Reference Screenshots

This directory contains user-supplied WakeTask alarm and competitor/context screenshots for reusable feature-iteration runs.

These files are intentionally checked into the workflow repo so Railway/Daytona runs can access the same visual context without relying on local machine paths.

Use these as reference material for:

- Apple Clock-like alarm list, edit, and create flows.
- Clear task/mission selection during alarm setup.
- Live alarm challenge and completion/share moments.
- Clean native iOS hierarchy without literal pixel copying.

See `manifest.json` for the source mapping and retention notes.
