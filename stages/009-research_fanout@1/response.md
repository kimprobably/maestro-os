### **Key Research Findings**

**Behavioral Frameworks Integrated:**
- **BJ Fogg's Behavior Model**: Motivation × Ability × Prompt (applied to WakeTask's two contexts: setup vs. morning)
- **James Clear's Habit Loop**: Cue → Craving → Response → Reward (alarm → desire to wake → missions → streak update)
- **Nir Eyal's Hook Model**: Trigger → Action → Variable Reward → Investment (accountability funnel)
- **Circadian Rhythm Research**: Peak grogginess 0–10 min post-wake; minimal cognitive capacity

**Critical Behavioral Gaps in Current Design:**
| Gap | Current State | Behavioral Problem |
|-----|---------------|-------------------|
| Jargon | Strictness/Wake Check/Modality unexplained | Low self-determined motivation |
| Immediate Feedback | Consistency % not visible to user | Weak reward reinforcement |
| Streak Visibility | Single % metric, no visual chain | Weak "don't break the chain" psychology |
| Escalation Framing | Red badge, no recovery path | Shame → abandonment |
| Morning Overload | Complex buttons, unclear state | Exceeds early-morning cognitive capacity |

**High-Priority Design Principles:**
1. **Clarity of purpose** (all UI must explain WHY)
2. **Progressive friction** (scale by strictness; Relaxed=1 mission, Strict=3 missions)
3. **Immediate, visible feedback** (streak counter updates on completion screen)
4. **Visual streak psychology** (day-streak counter + optional calendar heatmap)
5. **Accountability without shame** (reframe escalation as recovery opportunity)
6. **Setup-time guidance** (lock decisions during setup; morning UI automatic)
7. **Visual-first morning UI** (icons/emoji show modality instantly; 1–2 word labels)
8. **Accessibility as core** (color + icon + text; 44pt buttons; VoiceOver labels)

**Anti-Patterns to Avoid:**
- ❌ Generic productivity app UI (flat lists without motivation narrative)
- ❌ Jargon without explanation (strictness, modality, wake check)
- ❌ Punitive escalation framing (shame badges, no recovery path)
- ❌ Hidden button enable/disable rules (user confusion)
- ❌ Complex morning flows (exceeds cognitive capacity)
- ❌ Snooze buttons (defeats accountability)
- ❌ Dark patterns (manipulative notifications, forced sharing, fake urgency)
- ❌ Overlaid gamification (badges/XP without authenticity)
- ❌ Invasive camera usage (no consent, no privacy explanation)
- ❌ Competitor UI cloning (legal/ethical issues, misses unique angle)
