# Migrate to 1.0.0

## Summary
Initial release with unified authentication, two-tier documentation, and production-ready AI chat.

## New in 1.0.0
- Unified authentication (Apple + Google + Email) via Supabase
- Two-tier documentation system (Mintlify + /docs)
- LLM Prompt Packs for rapid customization
- Session persistence (users stay logged in)
- Dual chat UI styles (Bubble + Centered)
- 5 built-in themes
- 338+ tests with 85-90% coverage

## Breaking changes
None (initial release)

## Migration steps
1. Clone the repo
2. Copy `Config/Secrets.example.xcconfig` to `Config/Secrets.xcconfig`
3. Fill in your Supabase and RevenueCat credentials
4. Deploy Supabase Edge Function for AI
5. Run the app

See: `BUILDING_YOUR_APP.md` for complete setup guide.
